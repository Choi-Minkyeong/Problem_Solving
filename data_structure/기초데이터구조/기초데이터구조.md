## ✍ 기초데이터구조

### Array
- 순차 기억장소에 할당된 유한 개수의 동일 자료형 데이터 원소들
- size, isEmpty : O(1)
- get, set : O(1)
- add, remove : O(n)
- addFirst, removeFirst : O(n)
- addLast, removeLast : O(1)

### Linked list
- 동적메모리에 할당된, 링크에 의해 연결된 유한 개수의 데이터 원소 노드들
- size, isEmpty : O(1)
- get, set : O(n)
- add, remove : O(n)
- addFirst, removeFirst : O(1)
- addLast, removeLast : O(1)
